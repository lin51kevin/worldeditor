
### 存在的问题

| # | 问题 | 影响 |
|---|------|------|
| 1 | 右侧面板过于拥挤 | Property + Template + Tool 三面板挤在 280px 宽度内 |
| 2 | 工具栏信息过载 | 同时展示 6 组功能按钮，认知负担重 |
| 3 | 面板不可折叠/调整 | 左右面板固定宽度，无法拖拽或隐藏 |
| 4 | 缺少层级导航 | 没有面包屑或路径指示当前选中对象的层级 |
| 5 | 模板使用 emoji 图标 | Unicode 字符（━ ═ ≡ ☰ ⋔）不够专业 |
| 6 | 无命令面板 | 缺少 Ctrl+K 快捷搜索所有命令的能力 |
| 7 | 视口缺少 HUD | 无视角立方体、缩放控件、坐标轴指示器 |

## 方案 A：Studio —— 沉浸式暗色主题

> **设计灵感**：Blender 3.x + Unity 2023
>
> **核心理念**：以 3D 视口为绝对主角，所有 UI chrome 退到最低存在感。通过 Activity Bar + 浮出面板实现按需操作，最大化视口面积。

### 布局图

```
┌──────────────────────────────────────────────────────────┐
│  ◆ WE │ File  Edit  View  Tools │         project.xodr • │
├──┬───────────────────────────────────────────┬───────────┤

│🔍│    ╭───────────────────────────────╮       │ INSPECTOR │
│  │    │ ◇Select  ─Road  ═Lane  ⊕Junc │       │           │
│📐│    ╰───────────────────────────────╯       │ ▸ Basic   │
│  │                                           │   ID      │
│📦│              3D / 2D                      │   Name    │
│  │              VIEWPORT                     │   Length  │
│🔌│              ≈ 75% 面积                    │           │
│  │                                           │ ▸ Geometry│
│⚙ │                          ┌──────┐         │   #1 Line │
│  │                          │ View │         │   #2 Arc  │
│  │                          │ Cube │         │           │
```

### 组件说明

#### Activity Bar（左侧图标条，40px）

替代当前的固定 LayerPanel，使用纯图标条：

| 图标 | 功能 | 行为 |
|------|------|------|
| 🔍 | 场景树 / Outliner | 点击 → 从左侧浮出场景树面板 |
| 📐 | 道路编辑工具 | 点击 → 浮出工具面板 |
| 📦 | 模板库 | 点击 → 浮出模板选择面板 |
| 🔌 | 插件管理 | 点击 → 浮出插件管理器 |
| ⚙ | 设置 | 点击 → 浮出设置面板 |

- 浮出面板为 **overlay 层**，不推挤视口
- 点击视口或再次点击图标 → 面板自动收起
- 浮出面板带半透明毛玻璃背景

#### 浮动工具栏（Floating Toolbar）

```
╭─────────────────────────────────────────────╮
│  ◇ Select │ ─ Road │ ═ Lane │ ⊕ Junction   │
╰─────────────────────────────────────────────╯
```

- 居中悬浮于视口顶部
- 半透明毛玻璃背景（`backdrop-filter: blur(12px)`）
- 圆角胶囊形状（`border-radius: 20px`）
- 仅显示图标 + tooltip，无文字标签
- 当前激活的模式高亮显示

#### Inspector 面板（右侧，可折叠）

合并 PropertyPanel + TemplatePanel + ToolPanel 为统一面板：

```
┌─ INSPECTOR ──────────────────┐
│                              │
│ ▾ Basic Properties           │
│ ┌──────────────────────────┐ │
│ │ ID        road_001       │ │
│ │ Name      [Main Street ] │ │
│ │ Length    100.00 m       │ │
│ │ Junction  —              │ │
│ └──────────────────────────┘ │
│                              │
│ ▸ Geometry (3 segments)      │
│ ▸ Lanes (2 sections)         │
│ ▸ Elevation (1 record)       │
│                              │
│ ── ── ── ── ── ── ── ── ──  │
│                              │
│ ▸ Templates                  │
│ ▸ Tools                      │
└──────────────────────────────┘
```

- 可折叠的 Accordion 分组
- 面板宽度可拖拽调整（240px ~ 400px）
- 按 `N` 键切换显示/隐藏（类似 Blender N-Panel）

#### 视口 HUD 覆盖层

| 位置 | 元素 | 说明 |
|------|------|------|
| 右上角 | ViewCube | 3D 视角立方体，点击切换正交视图 |
| 右下角 | Zoom Slider | 垂直缩放滑杆 |
| 左下角 | 坐标轴指示器 | XYZ 三轴方向小部件 |
| 左上角 | 模式提示 | 当前编辑模式名称 + 快捷键提示 |

#### Command Palette

- 快捷键 `Ctrl+K` 打开
- 搜索所有命令（新建、打开、保存、撤销、视图切换等）
- 模糊搜索匹配
- 显示对应快捷键

### 优势与局限

| ✅ 优势 | ⚠️ 局限 |
|---------|---------|
| 视口面积最大化（~75%+） | Activity Bar 需要额外学习 |
| 按需浮出面板，不占固定空间 | 浮出面板遮挡视口内容 |
| 统一 Inspector 替代三个分散面板 | 毛玻璃效果可能影响低端 GPU 性能 |
| 浮动工具栏现代感强 | 不适合频繁切换多个面板的场景 |
| Command Palette 提升效率 | |
| ViewCube 提升 3D 导航体验 | |

## 方案 B：Canvas —— 极致留白

> **设计灵感**：Figma + Linear + Notion
>
> **核心理念**：把 OpenDRIVE 编辑器变成无边画布体验。UI chrome 退到几乎不可见，一切为视口让路。适合轻量编辑和演示展示场景。

### 布局图

```
┌──────────────────────────────────────────────────────────┐
│                                                          │
│  ☰                    project.xodr •               ⚙ 👤  │
│                                                          │
│  ┌──────┐                                                │
│  │Layers│     ╭─────────────────────────────────╮        │
│  │ 🔍   │     │  ◇  ─  ═  ⊕   浮动工具胶囊栏    │        │
│  │      │     ╰─────────────────────────────────╯        │
│  │▸Road1│                                                │
│  │ Road2│                                                │
│  │ Road3│            V I E W P O R T                     │
│  │      │            全屏画布 ≈ 95% 面积                   │
│  │      │                                                │
│  │      │                                                │
│  │      │                                  ┌───────────┐ │
│  │      │                                  │ Quick     │ │
│  │      │        选中 Road1 ──────────→     │ Inspector │ │
│  │      │                                  │           │ │
│  │      │     ╭──────────────────╮         │ Name: __  │ │

│                                                          │

└──────────────────────────────────────────────────────────┘
```

### 组件说明

#### 全屏画布（Full-Bleed Viewport）

- 视口占据 **整个窗口**，无固定边框或分隔线
- 所有 UI 元素浮动在视口之上（`position: absolute`）
- 视口背景色渐变过渡到 UI 元素，避免硬边

#### 极简顶栏

```
┌──────────────────────────────────────────────────────────┐
│  ☰                    project.xodr •               ⚙ 👤  │
└──────────────────────────────────────────────────────────┘
```

- 左侧：`☰` 汉堡菜单（点击展开全部菜单：文件 / 编辑 / 视图 / 工具）
- 中间：项目名称 + 修改指示器（`•`）
- 右侧：设置齿轮 + 用户头像
- 节省 28px 垂直空间（无传统菜单栏）
- 半透明背景，视口内容隐约可见

#### 浮动图层面板（左侧）

```
┌──────┐
│Layers│  ← 标题栏（可拖拽移动整个面板）
│ 🔍   │  ← 搜索过滤
│      │
│▸Road1│  ← 展开/折叠
│ Road2│  ← 单击选中，双击重命名
│ Road3│
│      │
└──────┘
```

- 半透明毛玻璃背景（`backdrop-filter: blur`）
- 可通过拖拽标题栏移动位置
- 默认显示在左侧；可拖到右侧或底部
- 按 `L` 键切换显示/隐藏
- 支持搜索过滤
- 列表项支持拖拽排序

#### Quick Inspector（右下浮出卡片）

选中道路/对象时自动在右下角浮出：

```
┌─ Quick Inspector ────────────┐
│  Road_001                    │
│                              │
│  Name     [Main Street    ]  │
│  Length   100.00 m           │
│  Lanes    2L + 2R            │
│  Type     Line → Arc         │
│                              │
│              [展开全部 ▸]     │
└──────────────────────────────┘
```

- 卡片式设计：圆角（12px）+ 阴影 + 毛玻璃
- 只显示最常用属性（5-6 项）
- 点击「展开全部」→ 右侧 slide-in 打开完整属性面板
- 取消选择 → 自动消失并带淡出动画
- 可拖拽移动位置

#### Context Toolbar（上下文操作栏）

选中元素时，在元素上方浮出迷你工具栏：

```
╭───────────────────────────────╮
│  🗑 删除  📋 复制  🔄 翻转   │
│  ➕ 添加车道  ✂ 分割道路      │
╰───────────────────────────────╯
```

- 仅在选中元素时出现
- 位置跟随选中元素（元素上方偏移）
- 不选中时完全隐藏
- 类似 Figma 选中后出现的操作栏

#### 浮动状态信息

```
左下角:  X: 123.4  Y: 789.0     ← 世界坐标
右下角:  Zoom: 100%              ← 缩放级别
```

- 半透明文字直接显示在视口上
- 不干扰视口内容
- 无背景色块

### 优势与局限

| ✅ 优势 | ⚠️ 局限 |
|---------|---------|
| 视口面积极致（~95%） | 频繁属性编辑效率不如固定面板 |
| UI 几乎不可见，沉浸感强 | 复杂编辑需多次点击展开完整面板 |
| Quick Inspector 按需浮出 | 浮动面板位置可能遮挡关键元素 |
| Context Toolbar 就近操作 | 对新手不够明显（UI 太隐蔽） |
| 极致留白和呼吸感 | 大量道路时图层面板不够高效 |
| 适合演示和展示 | 移动端适配性虽好但非核心场景 |

> **核心理念**：在保持专业工程软件信息密度的同时，引入现代设计语言。卡片式属性分组 + 上下文工具栏 + 可调面板。适合需要高频属性编辑的专业用户。

### 组件说明

#### 紧凑菜单栏

```
┌──────────────────────────────────────────────────────────┐
│  ◆ WE │ File  Edit  View  Tools  Help │     ☀/🌙  ─ □ ✕ │
└──────────────────────────────────────────────────────────┘
```

- 保留传统菜单栏（工程师熟悉的模式）
- 高度压缩到 28px
- 右侧集成主题切换 + 窗口控件
- 菜单项 hover 展开下拉（当前行为保持）

#### 上下文工具栏（Context Toolbar）

```
┌──────────────────────────────────────────────────────────┐
│  [🔍 Select ▾] [📐 Road ▾] │ ⟲ ⟳ │ 🗑 │  project.xodr •│
└──────────────────────────────────────────────────────────┘
```

- 替代当前的 40px 满功能 Toolbar
- **左侧**：编辑模式选择器（带下拉子工具）
  - `🔍 Select ▾` → 下拉展开：选择 / 框选 / 按类型选择
  - `📐 Road ▾` → 下拉展开：绘制道路 / 延伸道路 / 分割道路
- **中间**：操作按钮（撤销、重做、删除 — 仅图标）
- **右侧**：项目名称 + 修改指示器
- 根据当前编辑模式动态更新可用工具

#### Navigator 面板（左侧上半部分）

```
┌─ NAVIGATOR ──────────────────┐
│  🔍 搜索...                   │
│                              │
```

## 推荐策略与实施计划

### 推荐：方案 C 为主 + A/B 精华融合

综合考虑改造成本、专业用户需求、与现有代码的兼容性，建议：

1. **基础布局**：采用方案 C 的三栏结构（Navigator | Viewport | Inspector）
2. **Inspector 设计**：采用方案 C 的卡片式属性分组
3. **借鉴方案 A**：加入 Command Palette（Ctrl+K）+ 视口 HUD（ViewCube + 缩放滑杆）
4. **借鉴方案 B**：Quick Inspector 作为可选模式（设置中切换 固定面板 / 浮动卡片）
